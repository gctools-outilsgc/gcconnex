/**
 * Description
 */
Logo Manager lets you manage custom logos on your Elgg site. 

!!! This plugin requires the phloorFramework - 
please make sure you got the LATEST release !!!

This is a dependecny plugin for other menu related plugins.
Required plugins are:
- Logo for 1.8
- Favicon for 1.8
- Topbar Logo for 1.8

It combines the plugins mentioned above, creates a new admin menu 
section 'logo manager' and displays links to their settings.

Additionally administrators are given the control over the topbar
and footer logo to enable/disable them on demand.

/**
 * Description
 */
The phloor environment for your Elgg site.

Enables the core phloor environment. It provides basic functionalities, 
classes and views for all current and future phloor plugins. 

phloor plugins that use such functionality will declare this plugin as 
requirement. This plugin has to be above all other phloor plugins - 
phloorFramework itself does nothing.

!! NOTICE !!
Please consider, that this plugin is in BETA status. Moreover this means 
that every phloor plugin using this plugin is also beta.
